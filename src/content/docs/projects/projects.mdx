Interested in joining a current or future project? Join our Discord and message the project manager, or send a message to embedded@purdue.edu.  

We accept applications for new projects on a rolling basis. If you want to propose or lead your own, fill out the Project Proposal form (send a message to embedded@purdue.edu to access).

Please join the discord and reach out to a project manager if interested in joining an existing project!
## Future Projects

**Fall 2025**
- slayterHIL - Working to use Hardware-in-the-Loop testing to automatically validate embedded code on real hardware for faster, safer development.
- The Boiler Quad - The goal of The Boiler Quad is to help streamline disaster relief and rescue efforts while also gathering more information on the scene.
- Digital Operations Committee - Develop and maintain the website, tools, and docs
- Chariot Command partnership - Developing a project for our corporate partner
- FPGA Based Autotuner - This project focuses on making a 0-lag autotuner by building a dedicated, low-latency module on an FPGA for real-time performance.
- Agrovolo - Build an autonomous UAV that captures and processes field images to give farmers real-time insights into crop health and yield.
